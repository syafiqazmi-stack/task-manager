# Task Manager System

## Project Description
This project is a **Task Manager System** developed as part of an academic project.  
The system is designed to manage tasks between **Admin and Employees**, starting from task creation, assignment, monitoring, until task completion.

---

## Project Scope
The scope of this system includes:

- User authentication (Admin and Employee)
- Create tasks (Admin)
- View task list
- Edit tasks (Admin)
- Delete tasks (Admin)
- Assign tasks to employees
- Update task status (Employee)
- View task statistics

This project does not include payment processing, accounting, inventory management, or notification systems.

---

## Development Methodology
This project uses the **Waterfall methodology**.

The development process follows sequential phases:

1. Requirements analysis  
2. System design  
3. Implementation  
4. Testing  
5. Documentation  

Each phase must be completed before proceeding to the next stage.
